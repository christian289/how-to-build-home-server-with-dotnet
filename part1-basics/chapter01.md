# 제1장: 홈서버 입문

## 1.1 홈서버란 무엇인가?

홈서버(Home Server)는 가정 내에서 운영하는 개인용 서버 컴퓨터를 의미합니다. 클라우드 서비스에 의존하지 않고, 자신의 데이터와 서비스를 직접 관리하고 제어할 수 있는 강력한 도구입니다.

### 홈서버의 핵심 개념

**서버(Server)**란 다른 컴퓨터나 기기에 서비스를 제공하는 컴퓨터를 말합니다. 웹사이트를 호스팅하거나, 파일을 저장하거나, 미디어를 스트리밍하는 등 다양한 서비스를 24시간 제공할 수 있습니다.

홈서버는 다음과 같은 특징을 가집니다:

- **자율성**: 자신의 데이터를 완전히 통제하고 관리
- **확장성**: 필요에 따라 기능을 자유롭게 추가
- **경제성**: 초기 투자 후 월 구독료 없이 지속적으로 사용
- **학습 기회**: 리눅스, 네트워킹, 시스템 관리 등을 실습하며 학습

### 왜 홈서버를 구축해야 할까?

현대 사회에서 우리는 많은 클라우드 서비스를 사용합니다. Google Drive, Dropbox, Netflix, Spotify 등 편리한 서비스들이지만, 다음과 같은 한계가 있습니다:

1. **개인정보 보호**: 내 데이터가 타사 서버에 저장됨
2. **비용 증가**: 매월 구독료가 누적되면 상당한 금액
3. **서비스 종속성**: 서비스가 중단되거나 정책이 변경되면 대안이 없음
4. **기능 제한**: 제공자가 허용하는 기능만 사용 가능

홈서버를 통해 이러한 문제를 해결하고, 자신만의 프라이빗 클라우드를 구축할 수 있습니다.

## 1.2 N100 미니 PC가 홈서버로 적합한 이유

[Intel N100](https://www.intel.com/content/www/us/en/products/sku/231803/intel-processor-n100-6m-cache-up-to-3-40-ghz/specifications.html) 프로세서는 2023년에 출시된 저전력 CPU로, 홈서버 구축에 이상적인 선택입니다.

### 저전력 운영의 경제성

N100 프로세서의 TDP(Thermal Design Power)는 6W에 불과합니다. 이는 24시간 365일 가동해도 전기료 부담이 적다는 의미입니다.

**전기료 계산 예시**:
- N100 미니 PC 평균 소비전력: 약 10-15W (전체 시스템)
- 월간 전력 소비: 15W × 24시간 × 30일 = 10.8kWh
- 월 전기료 (kWh당 300원 기준): 약 3,240원
- 연간 전기료: 약 38,880원

반면, 일반 데스크톱 PC를 홈서버로 사용할 경우:
- 평균 소비전력: 100-150W
- 월간 전력 소비: 100W × 24시간 × 30일 = 72kWh
- 월 전기료: 약 21,600원
- 연간 전기료: 약 259,200원

N100 기반 홈서버는 일반 PC 대비 **연간 약 22만원의 전기료를 절감**할 수 있습니다.

### 24시간 운영에 최적화된 설계

N100 프로세서는 [Intel 7 공정](https://www.intel.com/content/www/us/en/newsroom/news/intel-accelerated-offensive-process-roadmap-2021.html)으로 제조되어 전력 효율과 안정성이 뛰어납니다.

주요 사양:
- **코어**: 4코어 4스레드
- **기본 클럭**: 0.8GHz
- **최대 터보 클럭**: 3.4GHz
- **내장 그래픽**: Intel UHD Graphics (24 EU)
- **메모리 지원**: DDR4-3200, DDR5-4800
- **최대 메모리**: 16GB

이 사양은 다음과 같은 작업을 동시에 처리하기에 충분합니다:
- 10개 이상의 Docker 컨테이너 실행
- Plex/Jellyfin 미디어 서버 (하드웨어 트랜스코딩 지원)
- Nextcloud 파일 서버
- 홈 오토메이션 (Home Assistant)
- 경량 웹 애플리케이션 호스팅

### 소음과 발열 관리

저전력 설계 덕분에 대부분의 N100 미니 PC는 **팬리스(Fanless) 설계** 또는 매우 조용한 쿨링 솔루션을 채택합니다.

**팬리스 설계의 장점**:
- 완전 무소음 운영
- 먼지 유입 최소화
- 기계적 고장 위험 감소
- 24시간 가동 시에도 소음 걱정 없음

발열 또한 매우 낮아서 일반적으로 40-50°C 정도의 온도를 유지하며, 실내 온도에 거의 영향을 주지 않습니다.

## 1.3 홈서버로 할 수 있는 일들

홈서버를 구축하면 상상 이상으로 많은 일을 할 수 있습니다. 다음은 실제로 구현 가능한 서비스들입니다.

### 개인 클라우드 스토리지
- **Nextcloud**: Google Drive/Dropbox 대체, 파일 동기화 및 공유
- **Syncthing**: P2P 방식의 안전한 파일 동기화
- **PhotoPrism/Immich**: Google Photos 대체, AI 기반 사진 관리

### 미디어 센터
- **Jellyfin/Plex**: 개인 Netflix, 영화/TV 프로그램 스트리밍
- **Navidrome**: Spotify 대체, 음악 스트리밍 서버
- **Calibre-Web**: 전자책 라이브러리 관리

### 생산성 도구
- **Vaultwarden**: 비밀번호 관리 (Bitwarden 호환)
- **WikiJS/BookStack**: 개인 위키, 지식 관리 시스템
- **Joplin Server**: Evernote 대체, 노트 동기화
- **Vikunja**: Todoist 대체, 할 일 관리

### 홈 오토메이션
- **Home Assistant**: 스마트홈 허브, 모든 IoT 기기 통합 제어
- **Node-RED**: 시각적 자동화 플로우 구성
- **Frigate**: AI 기반 보안 카메라 시스템

### 개발 환경
- **Gitea**: 개인 GitHub, Git 저장소 호스팅
- **code-server**: 웹 기반 VS Code, 어디서나 코딩
- **Portainer**: Docker 컨테이너 GUI 관리
- **PostgreSQL/MySQL**: 데이터베이스 서버

### 네트워크 서비스
- **Pi-hole**: 광고 차단 DNS 서버
- **WireGuard/Tailscale**: 안전한 VPN 접속
- **Nginx Proxy Manager**: 리버스 프록시, 도메인 관리

### 모니터링 및 관리
- **Grafana + Prometheus**: 시스템 모니터링 대시보드
- **Uptime Kuma**: 서비스 가동 상태 모니터링
- **Netdata**: 실시간 성능 모니터링

### AI 및 고급 기능
- **Ollama**: 로컬 LLM 실행 (ChatGPT 대체)
- **Stable Diffusion**: AI 이미지 생성
- **Whisper**: 음성-텍스트 변환

## 1.4 필요한 준비물과 예산 계획

홈서버를 구축하기 위해 필요한 하드웨어와 예상 비용을 정리했습니다.

### 최소 구성 (약 30-40만원)

| 항목 | 사양 | 예상 가격 |
|------|------|-----------|
| N100 미니 PC | N100, 8GB RAM, 128GB SSD 내장 | 250,000원 |
| 외장 SSD/HDD | 1TB SSD 또는 2TB HDD | 80,000원 |
| **합계** | | **330,000원** |

이 구성으로도 충분히 많은 서비스를 운영할 수 있습니다.

### 권장 구성 (약 50-70만원)

| 항목 | 사양 | 예상 가격 |
|------|------|-----------|
| N100 미니 PC | N100, 16GB RAM, 256GB SSD 내장 | 350,000원 |
| 추가 SSD | 2TB NVMe SSD (데이터 저장용) | 150,000원 |
| 외장 HDD | 4TB 외장 HDD (백업용) | 100,000원 |
| **합계** | | **600,000원** |

### 최상급 구성 (약 100-120만원)

| 항목 | 사양 | 예상 가격 |
|------|------|-----------|
| N100 미니 PC | N100, 16GB RAM, 512GB SSD 내장, 듀얼 NIC | 450,000원 |
| 대용량 SSD | 4TB NVMe SSD | 300,000원 |
| NAS HDD | 2 × 4TB NAS 등급 HDD (RAID 1) | 300,000원 |
| UPS | 무정전 전원장치 (500VA) | 100,000원 |
| **합계** | | **1,150,000원** |

### 추가 고려사항

**네트워크 장비**:
- 기존 공유기를 사용해도 되지만, 고급 기능을 원한다면:
  - Gigabit 공유기 (5-10만원)
  - 관리형 스위치 (10-20만원)

**소프트웨어**:
- 대부분 오픈소스/무료 소프트웨어 사용
- 도메인 비용: 연간 1-2만원 (선택사항)
- Cloudflare: 무료 (CDN 및 DNS)

**운영 비용**:
- 전기료: 월 3,000-5,000원
- 인터넷: 기존 인터넷 사용 (추가 비용 없음)
- 도메인: 연 15,000원 (선택사항)

### 단계별 투자 전략

초보자에게는 단계적으로 투자하는 것을 추천합니다:

**1단계 (초기)**: 기본 미니 PC + 외장 HDD (30-40만원)
- 기본 서비스 구축 및 학습
- Docker, Nextcloud, Jellyfin 등 체험

**2단계 (3-6개월 후)**: 스토리지 확장 (10-20만원)
- 추가 SSD/HDD 구매
- 데이터 증가에 대응

**3단계 (6-12개월 후)**: 고급 기능 (10-30만원)
- UPS 추가 (정전 대비)
- 네트워크 장비 업그레이드
- 추가 하드웨어 (스마트홈 기기 등)

이러한 단계별 접근을 통해 부담 없이 홈서버를 시작하고, 필요에 따라 확장할 수 있습니다.
